# Signal Weighting Core

Minimal TypeScript scoring core for weighting signals across structured review inputs.

## Purpose

Signal Weighting Core provides a compact reference implementation for assigning weighted scores to analytical signals such as relevance, confidence, source quality, and decision impact.

The project is designed as a small technical artifact for review workflows, AI-assisted evaluation, and structured decision-support prototypes.

## Core Concept

Each signal receives:

- a label
- a score
- a weight

The weighted value is calculated by multiplying the signal score by its assigned weight.

## Example Signals

- Relevance
- Confidence
- Source Quality
- Decision Impact
- Review Priority
